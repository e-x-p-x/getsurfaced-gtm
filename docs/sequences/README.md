Signal-triggered outreach sequences. Each file contains a complete
multi-channel cadence ready for execution.

## How to Use

1. Identify the buying signal
2. Open the matching sequence
3. Replace `{{first_name}}` with the prospect's name
4. Fill in any signal-specific details (competitor name, job title, content piece referenced)
5. Execute the cadence day by day
6. Track opens, replies, and meetings booked

## Customization

- Swap subject lines using the A/B variants provided
- Adjust timing (keep minimum 2-day gaps between touches)
- Add account-specific details from `docs/accounts/` if available
- For agency personas, emphasize multi-client workspace and white-label angles
- For freelancer personas, emphasize affordable pricing and professional reporting

## Signal Priority

Based on the ICP buying signals (see `docs/icp.md`):

1. **GEO Visibility Gap** — Strongest signal. Use when you have concrete proof the prospect is invisible in AI search.
2. **Hiring Signal** — Moderate signal. Best 30–60 days before the new hire starts evaluating tools.
3. **Content Engagement** — Warm signal. Best within 48 hours of the engagement to stay relevant.
